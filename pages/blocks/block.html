Неподвижный блок

Дана одного блока: неподвижного. Введите значения массы каждого блока, чтобы посмотреть на изменения. Примечание массы по умолчанию: 10кг. и 10кг.

Справочник

Блок — простое механическое устройство, позволяющее регулировать силу. Представляет собой колесо с жёлобом по окружности, вращающееся вокруг своей оси: жёлоб предназначен для каната, цепи, ремня и т. п. Неподвижный блок — ось блока в обоймах, закреплённых на балке или стене: меняет направление. При отсутствии трения для подъёма нужна сила, равная весу груза.
